Installing DOOM (1993) Mods
Introduction What you will need Option One Option Two Option Three Notes
Introduction

Here is a tutorial on how to install mods for the classic 1993 DOOM video game.

What you will need

Here is a list of what you'll need:

  • a PC
  • an IWAD (DOOM.WAD, DOOM2.WAD etc)
  • relevant mods to install in .WAD or .PK3 format (more on this later)
Option One

This is the "Drag and Drop" install method. It is the simplest method but is not recommended if you have a lot of mods or if you need to configure the load order (some mods may not work depending on the order in which they are loaded).

If you are only using one or two modifications (eg Brutal Doom), this method should be sufficient.

  1. Go to https://zdoom.org/downloads to download the latest version of GZDoom

  2. Extract and open the GZDoom folder

  3. Place your IWAD files (Doom.wad, Doom2.wad, Plutonia.wad, etc.) into the GZDoom folder

  4. Select all the files you want to use (Ctrl + Left Mouse Button) and drag them over the gzdoom.exe

Option Two

This is the "Config File" install method. This is a good way to manage your mod load order without using a launcher, though it is more technical than the other methods.

  1. Go to https://zdoom.org/downloads to download the latest version of GZDoom

  2. Extract and open the GZDoom folder

  3. Place your IWAD files (Doom.wad, Doom2.wad, Plutonia.wad, etc.) into the GZDoom folder

  4. Run gzdoom.exe

  5. This will create a gzdoom.ini file in your GZDoom folder

  6. Open the gzdoom.ini with Notepad

  7. Scroll down to the [Global.Autoload] section and type in Path=$PROGDIR/ and then the rest of the filename(s) for which file(s) you want to automatically load when GZDoom runs

  8. Save and exit from Notepad

  9. Whenever gzdoom.exe is ran, the files typed in the gzdoom.ini will be automatically loaded


    Example from Step 7:

    [Global.Autoload]
    Path=$PROGDIR/
    Project_Brutality_Test (10-1-17).pk3
    Path=$PROGDIR/PB_Allow_SV_Cheats.pk3
    Path=$PROGDIR/Doom_Metal_Vol_4.wad
Option Three

This is the "ZDL" install method. This is a good way to manage your mod load order by using the ZDL launcher. This method is recommended for all users.

  1. Go to https://github.com/qbasicer/qzdl to download the latest version of ZDL

  2. Extract and open the ZDL folder

  3. Run ZDL.exe

  4. Click on the Settings tab

  5. Under the Source Ports/Engines section, click the Add button and then browse for the gzdoom.exe

  6. Under the IWADs section, click the Add button and then browse for your IWAD files (Doom.wad, Doom2.wad, Plutonia.wad, etc.)

  7. Click on the Main tab

  8. Under the External Files section, click the Add button and then browse for any files you want to use when gzdoom.exe is ran

  9. Select which IWAD you want to play and then click on the Launch button

Notes

I have not had any issues with the load order that I use, but be aware that there is a supposed load order for files. If you're experiencing issues with your mods, they might be conflicting with one another because of the order in which they are being loaded. According to the Project Brutality v2.03a user manual, the load order should go like this (from top to bottom):

  • Map Packs, Megawads, Wads, etc.
  • Music Wads
  • Brutal Doom/Project Brutality
  • Compatability Patches
  • HUD Mods
  • Everything else
